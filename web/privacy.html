Privacy Policy

Last updated: February 2026

What is ebcli?

ebcli is a personal command-line tool that accesses bank account information (balances and transactions) through the Enable Banking API. It is used for personal financial management only.

Data collected

ebcli accesses the following data from your bank via Open Banking APIs:

  • Account identifiers (IBAN, account type)
  • Account balances
  • Transaction history

How data is processed

All data is processed locally on your own device. No bank data is transmitted to, stored on, or processed by any external server. The data flows directly from your bank through the Enable Banking API to your local machine.

Data storage

Bank data is not persisted. It is fetched on demand and exists only in memory during command execution. Connection metadata (session IDs, account aliases) is stored locally in a configuration file on your device with restricted file permissions.

Third parties

ebcli uses the Enable Banking API as an aggregation layer to access bank data via PSD2/Open Banking interfaces. No data is shared with any other third party.

Your rights

You can revoke access at any time by running ebcli disconnect or by revoking consent through your bank's online banking interface. You can delete all local configuration by removing ~/.config/ebcli/.

Contact
